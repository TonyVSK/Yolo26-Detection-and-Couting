# Yolo26-Detection-and-Couting
This repository contains all the files used to validate vehicle counts using deep learning models (in this case: yolo26).

## NECESSARY TO DOWNLOAD BEFORE EXECUTING INDEX.PY:
* yolo26l.pt Download it here: https://docs.ultralytics.com/pt/models/yolo26/
* video.mp4 (In this case, our .mp4 videos are hosted on Zenodo and they're private, but you can test this code with any .mp4 video and, of course, it will work as it should with public transport traffic videos).

## How to execute
* when u clone our repository, use the command
```
pip install -r requirements.txt 
```
* after that, just execute our index file
```
python.exe index.py
```
